## appium 활용한 쇼핑몰앱 테스트

- 쇼핑몰 앱 실행
- 국가를 “Angola”로 선택
- “Your name”은 빈칸으로 둠
- “Female”선택
- “Let’s Shop” 버튼 클릭
- 잘못된 입력시 토스트(Toast)오류 메세지가 적절히 표시되는지 확인 → assert 기능 확인해볼 수 있는 곳
- “Your name”에 “kim”입력
- 앱에서 스크롤해 “Air Jordan 9 Retro” 상품을 장바구니에 담기
- 장바구니 클릭
- “Air Jordan 9 Retro”  상품이 있는지 확인
- 하단의 “Please read our team of conditions”를 클릭
- 체크박스 클릭
- “Visit to the website to completes purchase”버튼 클릭
- 브라우저로 이동하는지 확인
- 백 버튼 클릭
- 첫 화면으로 이동했는지 확인
- 백 버튼 두번 클릭해 종료
